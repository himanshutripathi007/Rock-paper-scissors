Rock–Paper–Scissors (JavaScript)

A simple 5-round Rock–Paper–Scissors game built with JavaScript.
The player enters R, P, or S, and the computer randomly selects a move.
Final scores are shown after 5 rounds


How It Works

getHumanChoice() → asks user for R / P / S
getComputerChoice() → picks Rock, Paper, or Scissor randomly
playRound() → decides the winner of one round
playGame() → plays 5 rounds and shows final result


How to Run

Put the JavaScript code in script.js
Use this HTML:
<script src="script.js"></script>
Open the file in a browser
Open Console (F12 → Console)
Follow the prompts

Features

5 automatic rounds
Score tracking
Clear console output
Beginner-friendly logic
